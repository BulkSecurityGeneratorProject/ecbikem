Guidelines

1. Please return the bicycle to a designated NammaCycle Station within the stipulated time of the operation.
2. In case you are not able to return the bicycle, due to unavoidable circumstances, please drop the bicycle at the CiStup Station which will be open during the night.
3. Unreturned bicycles will be charged per day at Rs. 100 per day for the first 3 days; Rs. 500 for 6 days; and Rs. 1000 for over 7 days.
4. In case the User has not returned the Bicycle, Namma Cycle Team will call the User thrice and request him to return the bicycle. Complaints for missing bicycles will be booked at the end of 7 Days.
5. Repeated defaulters will be banned from using the system in future.

Safety First!

Obey Traffic Rules
Always ride in a single lane
Return before the first one hour
Use Hand Signals
Be careful while overtaking
Watch out for open car doors
Rest when tired
